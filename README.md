# how to use
```
git clone https://github.com/jp-north-man/nextlogin.git
cd nextlogin/
npm install @types/cookie-parser @types/bcrypt @types/jsonwebtoken jsonwebtoken @types/express express @types/cors @mui/material @emotion/react @emotion/styled
npm install concurrently ts-node @types/node --save-dev
npm run dev
```
